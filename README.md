# NyTimesMostPopular
This repository is for NY Times Most Popular android app

# Steps to build
1. Clone the repository on your system
2. Open Android Studio and import/open this project
3. Build the project by Build->Make Project or Build/Rebuild Project
4. Select the device/emulator to run the app
5. Run the project on device/emulator

#Steps to Run tests
1. Right click on android test files(ItemDetailFragmentTest, ItemListTest) or junit test file(FetchArticlesDataTest)
2. Then click on Run

# For generating code coverage
1. Right click on junit test file
2. More Run/Debug  then Run with code coverage
3. After the test is run, code coverage file is generated
4. Location for file <Project Folder> -> app > build > reports
